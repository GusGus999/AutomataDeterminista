Automata Finito Determinista
Cada case es un estado
Cada if es una transicion
Y un estado puede ser estado de aceptacion y esto se representa activando la variable booleana

La variable estado de tipo entero es el estado en el que se encuentra, iniciando por el estado 0
La variable indice representa la posicion que del caracter de la cadena que estamos recorriendo
lexema es la cantidad de simbolos aceptados por el analizador lexico y solo la usamos para tener algo que hacer en algunos casos
cadena es la cadena de entrada
